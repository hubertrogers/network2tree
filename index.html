This visualization shows movie recommendations (data from PopCha!) as a 'tree'. A link between two nodes indicates a recommendation, and the colour of a node represents its director.

Although a tree is not a natural representation for this structure (see a network visualization of the same data, or a matrix representation of analogous network data), it has great advantages in some use-cases.

Choose a new starting character-node by clicking a search result in the table.

Click on hidden nodes to see how the tree expands. Zoom by either (shift) double clicking on empty space, or by scrolling the mouse-wheel.

Built on free open-source web standards: CSS3, HTML5 and SVG using d3.js and Angular.js.

For best viewing we recommend Chrome, Firefox, Safari and IE >= 10 are also supported.
